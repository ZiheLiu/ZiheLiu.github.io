链接站内文章

参考 [GitHub] next 或者 hexo 链接站内文章的方法

对于链接站内文章有两种方式：

  • 不链接页内锚点 Hexo 语法支持 {% post_link %}。例如 {% post_link hexo-use Hexo 使用记录 %} ⇨ Hexo 使用记录。
  • 链接页内锚点 Hexo 语法不支持这种方式，只能自己手写链接了。例如 [2022/12/hexo-use/#运行](/2022/12/hexo-use/#运行) ⇨ 2022/12/hexo-use/#运行。
