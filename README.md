# ziheliu.github.io
`blog` 分支存储了 markdown 博客以及 hexo 配置。

## How to Run

### Run Server

```shell
$ hexo server
```

### Deploy to Remote Sites

```shell
# Generate static files firstly
$ hexo generate
# Deploy to remote sites
$ hexo deploy
```

### 修改样式
修改的样式在文件 `themes/next/source/css/_common/components/post/post-body.styl` 中。
