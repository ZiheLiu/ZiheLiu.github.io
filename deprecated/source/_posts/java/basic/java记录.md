# Java Bean

> [Java bean 是个什么概念]( https://www.zhihu.com/question/19773379/answer/40724984)

1、所有属性为 `private`。
2、提供默认构造方法。
3、提供 `getter` 和 `setter`。
4、实现 `Serializable` 接口。
