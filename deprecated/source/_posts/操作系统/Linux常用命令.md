# 文件相关

## ls

| 参数 | 英文                        | 解释                                                         |
| ---- | --------------------------- | ------------------------------------------------------------ |
| l    | using a long listing format | 输出长格式列表。                                             |
| h    | human-readable              | 与 `-l` 搭配使用，文件大小以可读的格式显示（1K、234M、5G）。 |
